# Time for Coffee API for iOS app


## Description

Based on [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash

docker   run -p 6455:6379 --rm -d redis:6-alpine
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
